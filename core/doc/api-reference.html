    Modules

    Opal

    Public API for the Opal coding agent SDK.

    Opal.Agent

    GenServer implementing the core agent loop.

    Opal.Agent.State

    Internal state for Opal.Agent.

    Opal.Auth

    Manages GitHub Copilot OAuth credentials.

    Opal.Config

    Typed configuration struct for Opal sessions.

    Opal.Config.Copilot

    Copilot-specific configuration: OAuth client ID and GitHub domain.

    Opal.Config.Features

    Unified feature toggles for optional Opal subsystems.

    Opal.Context

    Discovers project context files and agent skills from the filesystem.

    Opal.Events

    Registry-based publish/subscribe for session events.

    Opal.MCP.Bridge

    Bridges MCP tools into the Opal tool system.

    Opal.MCP.Client

    Anubis MCP client for connecting to external MCP servers.

    Opal.MCP.Config

    Discovers and parses MCP server configuration files.

    Opal.MCP.Resources

    Discovers and reads resources from connected MCP servers.

    Opal.MCP.Servers.Weather

    Example MCP server that provides weather information via stdio JSON-RPC.

    Opal.MCP.Supervisor

    Supervisor for MCP client processes within a session.

    Opal.Message

    A struct representing messages in an agent conversation.

    Opal.Model

    A struct representing model configuration for an agent session.

    Opal.Path

    Cross-platform path normalization and security utilities.

    Opal.Provider

    Behaviour for LLM provider implementations.

    Opal.Provider.Copilot

    GitHub Copilot provider implementation.

    Opal.RPC

    JSON-RPC 2.0 encoding/decoding. Transport-agnostic.

    Opal.RPC.Handler

    Dispatches JSON-RPC methods to Opal library functions.

    Opal.RPC.Protocol

    Declarative protocol specification for the Opal JSON-RPC 2.0 API.

    Opal.RPC.Stdio

    JSON-RPC 2.0 transport over stdin/stdout.

    Opal.Session

    GenServer managing a conversation tree with branching and persistence.

    Opal.Session.Compaction

    Context window compaction following pi's approach.

    Opal.SessionServer

    Per-session supervisor that owns the full session process tree.

    Opal.Skill

    Parses and validates Agent Skills following the agentskills.io specification.

    Opal.SubAgent

    Spawns and manages child agents that work in parallel.

    Opal.Tool

    Behaviour defining the interface for agent tools.

    Opal.Tool.Edit

    Applies a search-and-replace edit to a file.

    Opal.Tool.Read

    Reads file contents, optionally returning a slice of lines.

    Opal.Tool.Shell

    Runs shell commands cross-platform with timeout support.

    Opal.Tool.SubAgent

    Tool that allows an agent to spawn a sub-agent for delegated tasks.

    Opal.Tool.Tasks

    Project-scoped task tracker backed by DETS (built into Erlang).

    Opal.Tool.UseSkill

    Loads an agent skill's full instructions into the active context.

    Opal.Tool.Write

    Writes content to a file, creating parent directories as needed.
