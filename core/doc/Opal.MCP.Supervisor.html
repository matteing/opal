    Opal.MCP.Supervisor (Opal v0.1.0)

    Copy Markdown View Source

    Supervisor for MCP client processes within a session.

    Starts one Opal.MCP.Client child per configured MCP server using a :one_for_one strategy — each server connection is independent, so a crash in one doesn't affect others.

    Supervision tree placement

    SessionSupervisor (:rest_for_one)
├── Task.Supervisor      — tool execution
├── DynamicSupervisor    — sub-agents
├── Opal.MCP.Supervisor  — MCP clients
│   ├── Client :server_a
│   ├── Client :server_b
│   └── ...
├── Opal.Session         — persistence (optional)
└── Opal.Agent           — the agent loop

    When the session shuts down, this supervisor cascades termination to all Anubis client processes, which cleanly close their connections.
