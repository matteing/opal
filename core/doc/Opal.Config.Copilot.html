    Copilot-specific configuration: OAuth client ID and GitHub domain.

    Fields

    • :client_id — OAuth App client ID for the Copilot device-code flow. Default: "Iv1.b507a08c87ecfe98" (the VS Code Copilot Chat extension's ID).

    • :domain — GitHub domain for authentication endpoints. Default: "github.com". Change for GitHub Enterprise Server instances.
