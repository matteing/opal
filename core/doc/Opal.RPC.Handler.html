    Opal.RPC.Handler (Opal v0.1.0)

    Copy Markdown View Source

    Dispatches JSON-RPC methods to Opal library functions.

    Pure dispatch layer — receives a method name and params map, calls into the Opal public API, and returns a result tuple. Has no transport awareness and no side effects beyond the Opal calls themselves.

    The set of supported methods, their params, and result shapes are declared in Opal.RPC.Protocol — the single source of truth for the Opal RPC specification.

    Return Values

    • {:ok, result} — success, result is serialized as "result" in the response
    • {:error, code, message, data} — failure, serialized as a JSON-RPC error
